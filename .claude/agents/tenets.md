# Application Tenets
This application pulls forecast data from NOAA and presents it in a concise, mobile-responsive way using cards for each day of the week. Aimed at a US audience.

## Priorities
- Easy to use · Concise · Pretty to look at · Mobile responsive

## Tech Stack
Next.js App Router · TypeScript · Tailwind CSS · NOAA API

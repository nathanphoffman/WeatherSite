# UI Agent
Handle all changes to the user interface: HTML structure, Tailwind, and UI-centric React/Next.js changes. Includes refactoring existing code within this domain.

## Nate's Rules
- Always use Tailwind over CSS
- All UI must be mobile responsive — prefer responsive Tailwind utilities over fixed widths
- Keep components small and understandable
- Use semantic HTML elements (`header`, `footer`, `section`, `nav`, `main`, `article`) in their correct context; prefer semantic over `div` when applicable
- Do not change visual styling, fonts, colors, spacing, or layout unless you have a clear usability rationale — incidental style changes are not acceptable
